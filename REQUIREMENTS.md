# AI Text Polisher 要件定義書 v3

## 1. プロジェクト概要

音声入力やラフなメモに含まれる「フィラー（えー、あの等）」「言い直し・訂正」を、
macOS標準STT＋ローカルLLM（Ollama）でリアルタイムに整形・ビジネス文章化するメニューバーアプリ。

**処理フロー:**
```
[⌘⌥K 長押し] → [マイク録音] → [SFSpeechRecognizer でSTT] → [Ollama で整形] → [自動貼り付け]
[⌘⌥K 短押し] → [クリップボード取得] → [Ollama で整形] → [自動貼り付け]
[テキスト選択 + ⌘⌥K 長押し] → [選択テキスト保存] → [音声で指示] → [Ollama で処理] → [自動貼り付け]
```

## 2. ターゲット環境

| 項目 | 要件 |
|------|------|
| OS | macOS 13以降（Apple Silicon / M1推奨）|
| RAM | 4GB以上（8GB以上推奨）|
| Python | 3.11以降 |
| STTエンジン | macOS SFSpeechRecognizer（Siri/Dictationと同エンジン）|
| LLMエンジン | Ollama（ローカルAPIサーバー）|
| 推奨モデル | `gemma3:4b`（標準）/ `gemma3:1b`（低スペック）|

## 3. 機能要件

### 3.1 コア機能

1. **音声入力モード（Push-to-Talk）**
   - `⌘⌥K` を0.5秒以上押し続けると録音開始（アイコンが🎙️に変化）
   - 離したタイミングでSFSpeechRecognizerでSTT処理
   - STT結果をOllamaで整形し、クリップボードに書き戻し自動ペースト

2. **テキスト整形モード**
   - `⌘⌥K` 短押しでクリップボードのテキストを取得・整形・貼り付け

3. **選択テキスト指示モード**
   - テキストを選択した状態で `⌘⌥K` 長押し
   - 選択テキストをコンテキストとして保存
   - 音声で指示（翻訳・要約・言い換え等）→ 結果を貼り付け

4. **Ollama自動起動**
   - アプリ起動時にOllamaが未起動なら自動で `ollama serve` を実行

5. **ログイン時自動起動**
   - LaunchAgentによりログイン時に自動起動（ターミナル不要）

### 3.2 整形機能

- フィラー除去（えー/あの/えっと/まあ/なんか等）
- 言い直し統合（最後の言い直しのみ採用）
- 重複除去
- 番号列挙の自動箇条書き変換（Python側で前処理）
  - 「1つ目/1点目/1軒目」等の誤認識も対応
- 英語の固有名詞・専門用語はそのまま英語で保持
- 自然なビジネス敬語（です・ます調）

### 3.3 UX/UI

- メニューバーアプリとして常駐（Dockに表示しない）
- アイコン: ✨（処理中: ⏳、録音中: 🎙️）
- メニューに現在の使用モデルを表示
- 完了時にmacOS通知でプレビュー表示

### 3.4 macOS権限要件

| 権限 | 用途 | 付与方法 |
|------|------|---------|
| アクセシビリティ | グローバルショートカット検知・自動ペースト | システム設定から手動付与 |
| マイク | 音声録音 | システム設定から手動付与 |
| 音声認識 | SFSpeechRecognizer | 初回起動時にダイアログ |

## 4. 技術スタック

| レイヤー | 採用技術 | 理由 |
|---------|---------|------|
| GUI / メニューバー | rumps | macOSメニューバーアプリに特化・軽量 |
| グローバルショートカット | NSEvent（PyObjC） | macOS 15対応・クラッシュしない |
| 音声録音 | sounddevice + numpy | シンプルなPython音声入力 |
| STT | SFSpeechRecognizer（PyObjC） | macOS標準・日本語高精度・モデルDL不要 |
| LLM整形 | Ollama REST API | ローカルLLM・プライバシー保護 |
| クリップボード | pyperclip | シンプル |
| 自動ペースト | osascript（AppleScript） | 安定動作 |
| 自動起動 | launchd（LaunchAgent） | macOS標準の自動起動機構 |

## 5. パフォーマンス目標

| 処理 | 目標時間 |
|------|---------|
| STT（SFSpeechRecognizer） | 1〜2秒 |
| LLM整形（gemma3:4b） | 3〜7秒 |
| 合計（音声入力→貼り付け） | 10秒以内 |

## 6. 設定（config.json）

```json
{
  "model": "gemma3:4b",
  "keep_alive": 300,
  "ollama_timeout": 30
}
```

| キー | 説明 |
|------|------|
| `model` | Ollamaモデル名 |
| `keep_alive` | モデルRAM保持秒数（0=即解放/-1=永続）|
| `ollama_timeout` | APIタイムアウト秒数 |

## 7. インストール方法

### エンドユーザー向け
`install.command` をダブルクリックするだけ。
Homebrew・Python・Ollama・パッケージ・モデルDL・自動起動設定まで全自動。

### 開発者向け
```bash
pip3.11 install -r requirements.txt
python3.11 app.py
```
